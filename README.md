# Alicheh's Digital Garden

This is the source code for my personal website and digital garden, hosted at [alicheh.github.io](https://alicheh.github.io).

## About
This site is a collection of my explorations, learnings, and projects, focusing on AI, software engineering, and the intersection of technology and society.

## Tech Stack
- **Framework:** [Hugo](https://gohugo.io/)
- **Theme:** [PaperMod](https://github.com/adityatelange/hugo-PaperMod)
- **Deployment:** GitHub Pages via GitHub Actions

## How to run locally
1. Clone the repository:
   ```bash
   git clone --recursive https://github.com/alicheh/alicheh.github.io.git
   ```
2. Install Hugo (extended version recommended).
3. Run the development server:
   ```bash
   hugo server
   ```
4. Open `http://localhost:1313` in your browser.
